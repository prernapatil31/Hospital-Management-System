The Hospital Management System is a full-stack web application developed using React, Spring Boot, and MySQL to efficiently manage hospital operations.

It allows users to handle patient records, doctor details, and appointment scheduling through a user-friendly interface.

The system is built with RESTful APIs, ensuring smooth communication between the frontend and backend, and follows proper database relationships using JPA.

This project demonstrates practical implementation of CRUD operations, API integration, and real-world problem solving in a healthcare management system.
